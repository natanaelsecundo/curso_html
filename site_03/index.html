O quesão tags?

TGS, ou "tags", em HTML são elementos que definem a estrutura e o conteúdo de uma página web.
Elas são usadas para criar diferentes tipos de conteúdo, como textos, links, imagens e tabelas.
Cada tag tem uma função específica, como"<h1>" para cabeçalhos ou "<p>" para parágrafos.
